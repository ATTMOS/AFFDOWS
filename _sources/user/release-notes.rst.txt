Release Notes
^^^^^^^^^^^^^

The new features released with each AFFDO version are as follows. 

AFFDO 24.01
***********

* Automated Force Field Development: Generates ligand-specific Generalized Amber Force Field torsion parameters.
* User-Friendly Interface: Jupyter-notebook based, widget powered, easy to use graphical user interface .
* Computer Hardware Flexibility: Cloud instances or local computer hardware can be conveniently integrated for computationally expensive tasks. 
* Supports Linux and MacOS operating systems.
